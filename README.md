# Space-Engineers-Discord-Bot
A discord bot thats by default pings a space engineers server (Keen NA1 by default) to get the playerlist.
# SEBot Dependencies
* Discord.py
* Pip install Discord.py
* aiohttp
* Pip install aiohttp
# Server.js Dependencies
* express
* npm install express
* gamedig
* npm install gamedig@4.0.0
* cors
* npm install cors
